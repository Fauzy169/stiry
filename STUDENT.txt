📘 Informasi Proyek Dicoding Story Platform
📝 Keterangan
Platform ini dikembangkan untuk memenuhi submission kelas Belajar Fundamental Aplikasi Android dari Dicoding.

Menggunakan OpenStreetMap (Leaflet) untuk tampilan peta, tanpa memerlukan API key.

Endpoint utama API: https://story-api.dicoding.dev/v1

🚀 Fitur Utama
✅ Autentikasi

Login dan Register pengguna

📜 Daftar Cerita

Menampilkan daftar cerita dari API

Setiap cerita berisi nama, deskripsi, foto, dan lokasi

➕ Menambah Cerita

Upload cerita baru dengan foto

Penentuan lokasi langsung dari peta

Dukungan kamera perangkat untuk ambil gambar

🗺️ Peta Interaktif

Cerita dapat dilihat di peta

Pemilihan titik lokasi cerita via klik pada peta

Mendukung multiple map layer (tile switching)

⭐ Favorit

Menambahkan dan menghapus cerita favorit

Disimpan di IndexedDB lokal

♿ Aksesibilitas

Skip to content

Label form & elemen semantik

💡 User Experience

Desain responsif di berbagai perangkat

Transisi antar halaman menggunakan View Transition API

Tampilan warna-warni (colorful & modern)
